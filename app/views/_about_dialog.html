This is a diploma project of Viktor Maksimenko, ONPU student.

This web app displays relations between themes learned at university and themes required by IT-companies. Setup interested universities and companies for building graph.

  • lens Red rectangles display chosen positions
  • lens Green rectangles display chosen university specialities
  • lens Yellow rectangles display themes. Node size depends on number of using

The graph is scalable and draggable. Click node to get info about it. Use export function to save graph as SVG, JPG, PNG, GIF or TIFF.

Sign up to have an ability to save your graph. If you are a representative of University ot IT-company, go on sign up and add you organisation to list

For graph drawing we use Cytoscape.js - Graph theory library for analysis and visualisation
